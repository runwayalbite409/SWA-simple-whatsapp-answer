## ⚙️ How to use the software

The application provides a main dashboard to manage your messaging tasks. The left sidebar contains buttons for your settings, message templates, and the message scheduler. 

### Setting up templates
1. Click the "Templates" button in the sidebar.
2. Select "New Template."
3. Type a name for your message.
4. Write the content you wish to send.
5. Click "Save."

### Creating a schedule
1. Go to the "Scheduler" tab.
2. Select "Add Task."
3. Choose the template you created earlier.
4. Set the date and time for the message delivery.
5. Provide the contact name or group name for the recipient.
6. Click "Confirm" to activate the task.

The application monitors the clock and triggers the message based on your programmed time. You can view past messages by selecting the "History" tab.

## 🛡️ Privacy and data protection

This application keeps your data inside your own computer. The database file saves your contact entries and message logs locally. No external servers scan your messages or personal details. You maintain full control over your information at all times. If you uninstall the application, the local database file remains in your folder. You can delete this file yourself to permanently remove your data.

## 💡 Troubleshooting common issues

Most users find the application easy to run. However, follow these tips if you face problems:

- Application will not start: Restart your computer and try to open the program again.
- Messages send late: Check your system clock to ensure the time matches your timezone settings.
- Connection errors: Verify your internet connection remains stable.
- Missing templates: Ensure you clicked the save button before you left the template editor.

If the software does not behave as expected, close the window and re-open it. This clears temporary issues with the internal system. Always use the latest version from the releases page to get the best performance.
