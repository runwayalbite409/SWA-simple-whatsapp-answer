# Simple WhatsApp Answer

Desktop application for Windows focused on WhatsApp message scheduling, contact management and template-based automated communication.

> ⚠️ Version `1.0.0` is the first public MVP (Minimum Viable Product) release.  
> Significant changes, UI improvements, new features and stability updates are planned for future versions.

---

## Features

- Message scheduling with date and time selection
- WhatsApp Web integration
- Contact management
- Dynamic message templates
- Real-time chat interface (No audio or images)
- Message history and logs
- Local data persistence
- Import/export database utilities

---

## Platform

Currently available only for:

- Windows 10
- Windows 11

- No external server is used for storing conversations or contacts
- WhatsApp sessions are stored locally using persistent authentication
